Overview
========

It is basic TCP client/server example with String content encoder/decoder.

Running
=======

To run the example execute:

```
$ cd RxNetty/rx-netty-examples
$ ../gradlew runTcpEchoServer
```

and in another console:

```
$ cd RxNetty/rx-netty-examples
$ ../gradlew runTcpEchoClient
```
